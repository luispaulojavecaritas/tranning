Titan Documentation

Installation

Titan provides a main template.xhtml and additional layoutmenu.xhtml, topbar.xhtml, footer.xhtml files for the base layout. These 4 files must be placed under WEB-INF folder. Provided empty-page.xhtml is a sample content page using the main template.xhtml that defines "content" as the main ui:define placeholder. Other required resources are the css, js, image and font files that are located inside resources/titan-layout folder, simply copy the titan-layout folder to your %WEB-APP-FOLDER%/resources folder so that final path would be %WEB-APP-FOLDER%/titan-layout/. Please refer to demo app or maven project of the demo app as the reference.

Theme is a regular PrimeFaces theme and setup is simple as defining primefaces.THEME context parameter in web.xml as "titan", refer to themes page for more information.

BoxedLayout Mode

Adding 'BoxedLayout' class to body tag applies a padding to the each side of the content to narrow down the interface instead of spanning 100% width.

Topbar Modes

Titan Layout has 2 different topbar modes, 'Fixed Topbar' and 'Flying Topbar'. In fixed mode, menu scrolls along with the page thus kept in viewport and in flying mode menu position is static.
Class names to apply the body tag are, 'FixedTopbar' or 'FlyingTopbar'

Important Note : You can use only one of these two classes .

Menu Modes

There are two alternatives for menu modes; WideMenu and SlimMenu.
Class names to apply the body tag are, 'WideMenu' and 'SlimMenu'

Important Note : You can use only one of these two classes .

Core Layout

Core Layout is a set of CSS classes used to create responsive grid layouts with additional utilities. Titan uses Core Layout internally, however for grid layout you may also choose another utility like Grid CSS or Bootstrap Grid.

Migration Guide
1.1.1 to 1.1.2
  • - Replace theme jar with the new jar.
  • - Replace TitanMenu.java with the new TitanMenu.java.
1.1 to 1.1.1
  • - Update titan-layout.css, titan-layout.less, core-layout.css and layout.js
  • - Replace theme jar with the new jar.
  • - Replace TitanMenuRenderer.java with the new TitanMenuRenderer.java.
  • - Replace TitanMenu.java with the new TitanMenu.java.
1.0.1 to 1.1
  • - Replace theme jar with the new jar.
1.0 to 1.0.1
  • - Update titan-layout.css, titan-layout.less, core-layout.css and template.xhtml
  • - Replace theme jar with the new jar.
  • - Replace TitanMenuRenderer.java with the new TitanMenuRenderer.java.
  • - Replace Titan font files with the new Titan font files.(titan-layout/fonts/*)
